# Proyecto de Registro de Empleados en PHP

Este proyecto es una aplicación básica en PHP que permite registrar el nombre de un empleado y su hora de entrada desde la consola.

## 📌 Funcionalidad

* Solicita el nombre del empleado
* Solicita la hora de entrada
* Muestra un mensaje confirmando el registro

## 🛠️ Tecnologías

* PHP (CLI)

## ▶️ Cómo ejecutar

1. Abre la terminal
2. Ubícate en la carpeta del proyecto:

   ```bash
   cd proyecto
   ```
3. Ejecuta el archivo:

   ```bash
   php RegistroEmpleados.php
   ```
